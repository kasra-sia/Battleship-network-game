# Battleship-network-game
My first network(client-server) project for Advance Programming homework

## Introduction
This project is a Battleship game with online multiplayer functionality and a leaderboard. Players can engage in battles with each other, view their scores, and spectate ongoing games.

## Features

- **Online Multiplayer**: Players can create or join game rooms to play Battleship against each other in real-time.
- **Leaderboard**: Track and display the scores of the top players.
- **Game Spectator**: Watch ongoing games between other players.
- **User Authentication**: Secure user registration and login system.
- **Game Logic**: Implement the core Battleship game logic, including ship placement and attack mechanisms.
- **Real-Time Communication**: Use WebSocket for real-time communication between players.
